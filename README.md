# Kanban Board

Private encrypted Kanban board desktop app.

## Requirements

Install **Node.js** (one-time): https://nodejs.org — download the LTS version and run the installer.

## Setup (run once after cloning)

```
npm install
```

## Run the app

```
npm start
```

## Build a Windows installer (optional)

```
npm run build:win
```

Outputs to `dist/` — use `Kanban Board Setup 1.0.0.exe` to install, or `Kanban Board 1.0.0.exe` to run portable with no install.
